# blog-api

# Starting the server
```bash
git clone https://github.com/PrathameshGandule/blog-api.git
```
```bash
cd blog-api-lite
pnpm install
```
- Setup .env configs
```bash
pnpm run build
```
```bash
pnpm run start
```
